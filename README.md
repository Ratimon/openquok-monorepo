# OpenQuok monorepo

**OpenQuok** is a controlled content pipeline that sits on top of your AI tools like **OpenClaw**—turning high-volume generation into high-quality output with a structured review + scheduling flow.

### Prerequisites

- Node.js 24+
- Corepack (for pnpm)

```bash
corepack enable
```

### Tech stack

- Supabase DB & Auth
- Express.js
- Svelte 5
- Tailwind CSS
- DaisyUI
- Redis
- Stripe
- Resend
- Sentry
- Vercel
